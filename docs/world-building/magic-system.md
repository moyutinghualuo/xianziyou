# 魔法体系

## 七大派系

### 1. 元素系（Elemental）
操控火、水、风、土四大元素。最古老也最基础的魔法派系。

- **火魔法**：破坏力最强，适合战斗
- **水魔法**：治愈与防御
- **风魔法**：速度与移动
- **土魔法**：防御与建造

### 2. 光明系（Light）
治疗、净化、防护。圣殿骑士团的核心力量。

### 3. 暗影系（Shadow）
隐匿、诅咒、精神控制。被主流社会排斥，但并非邪恶。

### 4. 自然系（Nature）
与动植物沟通，操控植物生长。精灵族的专属魔法。

### 5. 时空系（Spacetime）
极为罕见的天赋。可短暂扭曲空间、预知未来片刻。

### 6. 召唤系（Summoning）
从异界召唤生物或灵体。

### 7. 炼金系（Alchemy）
将魔力注入器物，制作魔法道具。

## 魔力等级

| 等级 | 名称 | 描述 |
|------|------|------|
| 1 | 学徒 | 刚觉醒，只能使用基础魔法 |
| 2 | 见习 | 掌握一到两个法术 |
| 3 | 正式 | 熟练运用一个派系 |
| 4 | 专家 | 精通一个派系，可自创法术 |
| 5 | 大师 | 跨派系融合，影响范围巨大 |
| 6 | 贤者 | 传说中的境界，可改写规则 |
