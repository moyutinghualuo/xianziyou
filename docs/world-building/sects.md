# 宗门大全

## 🏔️ 正道宗门

### 仙武宗
**最古老的正道宗门**

- 掌门：待定
- 弟子：数千人
- 传承：剑修为主
- 特色功法：《仙武剑经》
- 位置：中州·仙武峰

### 天玄宗
**以法阵著称**

- 特色：阵法与法宝
- 传承：阵法师培养
- 代表人物：待定

### 云海派
**云中之宗**

- 特色：御空飞行
- 传承：风属性功法
- 位置：中州·云海仙境

## 🔴 魔道宗门

### 血魔宗
**以血道著称**

- 特色：血祭、融合
- 传承：血魔秘法
- 名声：臭名昭著

### 吞天教
**掠夺一切**

- 特色：吞噬之道
- 传承：吞天秘法
- 行事：邪恶残暴

## ⚪ 中立势力

### 商会联盟
**以商业为主**

- 特色：交易与信息
- 传承：经营之道
- 影响力：遍布各州

### 狩猎者协会
**冒险者的组织**

- 特色：任务系统
- 传承：经验分享
- 成员：来自各地
