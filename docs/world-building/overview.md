# 世界观概览

欢迎来到仙侠世界的入门指南。这里汇聚了关于这个充满仙气的世界的所有基本信息。

## 🌍 世界基本信息

| 项目 | 描述 |
|------|------|
| **世界名称** | 待定 |
| **设定时代** | 修仙纪元 |

## 📜 历史时代

### 上古时代
- 仙魔战争
- 天地初分
- 各族独立

### 中古时代
- 宗门林立
- 魔道衰落
- 人族兴盛

### 当代
- 修仙路开放
- 各派争锋
- 故事开始

## 🏔️ 地理版图

- **中州** — 修仙者最多的大陆
- **北荒** — 妖兽聚集之地
- **西漠** — 秘境与古迹
- **南疆** — 蛮荒与神秘
- **东洲** — 海洋与传说
