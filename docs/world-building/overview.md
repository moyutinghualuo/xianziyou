# 世界观概览

## 世界背景

这个世界名为 **艾瑟兰（Aetherlan）**，是一片被魔力充盈的大陆。远古时代，众神在此创造了万物：

- **创世纪元** — 众神降临，塑造山川河流
- **黄金纪元** — 人类崛起，建立魔法文明
- **暗影纪元** — 魔神入侵，大陆分裂
- **新生纪元** — 当前时代，各方势力角逐

## 世界规则

1. 魔力来源于大地深处的 **源核**
2. 只有少数天赋者能够感知并操控魔力
3. 魔法分七大派系（详见 [魔法体系](magic-system.md)）

## 大陆地理

艾瑟兰大陆分为四大区域，详情见 [地理志](geography.md)。
