# 地理志

## 四大区域

### 北境 — 冰霜王国
- **首都**：永冬城
- **气候**：终年冰雪
- **特产**：寒铁、冰晶石
- **主要势力**：北境公爵领

### 东陆 — 太阳帝国
- **首都**：金辉城
- **气候**：温暖湿润
- **特产**：丝绸、香料、魔法卷轴
- **主要势力**：帝国皇室、魔法学院

### 南疆 — 翡翠联盟
- **首都**：绿茵城
- **气候**：热带雨林
- **特产**：灵药、魔兽材料
- **主要势力**：精灵族、德鲁伊议会

### 西荒 — 自由城邦
- **首都**：无（各城邦自治）
- **气候**：沙漠与草原交替
- **特产**：矿产、佣兵
- **主要势力**：商人公会、冒险者协会

## 重要地标

- **源核深渊**：大陆中心，魔力源头
- **龙脊山脉**：分割东西大陆的天险
- **迷雾海**：环绕大陆的未知海域
