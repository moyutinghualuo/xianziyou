# 地理志

## 🗺️ 五州版图

### 中州
**最繁华的修仙大陆**

- 面积：最大
- 人口：最多
- 宗门：最密集
- 特点：中原腹地，修仙资源丰富

主要城市：
- 宗门联盟城
- 交易中心城
- 传承塔

### 北荒
**妖兽与冰霜的国度**

- 气候：严寒
- 生物：妖兽众多
- 资源：稀有矿物、寒冰精髓
- 特点：危险与机遇并存

主要地点：
- 妖兽森林
- 冰川之心
- 北极秘境

### 西漠
**古迹与秘密的沙漠**

- 气候：炎热干燥
- 特点：上古文明遗迹
- 资源：古宝、秘籍
- 危险：诅咒、傀儡

主要遗迹：
- 沙漠古城
- 金字塔遗迹
- 秘藏洞窟

### 南疆
**蛮荒与神秘之境**

- 气候：炎热多雨
- 生物：蛮族、异种
- 特点：原始力量强大
- 资源：宝药、魂石

主要势力：
- 蛮族部落
- 兽人王庭
- 邪教组织

### 东洲
**海洋与岛屿的传说**

- 特点：多岛屿、多海上势力
- 资源：珍珠、海兽材料
- 文明：海上修仙宗门

主要区域：
- 东洋诸岛
- 深海秘境
- 航海贸易路线

## 🏯 著名地点

### 禁地

!!! danger "三大禁地"
    1. **灭世遗迹** — 上古战场，能量混乱
    2. **堕渊之下** — 魔气汇聚，生人勿进
    3. **时空裂缝** — 规则混乱，坐标不定

### 圣地

!!! success "五大圣地"
    1. **仙武峰** — 最高的山峰，灵气最浓
    2. **悟道湖** — 适合顿悟的圣地
    3. **生死谷** — 历练者的必经之地
    4. **长生林** — 神秘的古老森林
    5. **天门关** — 通往仙界的入口
