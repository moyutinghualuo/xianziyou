# 人物关系

## 核心关系

- **艾琳 ↔ 凯恩**：保护者与被保护者。凯恩最初只当是任务，逐渐产生超越职责的感情。
- **艾琳 ↔ 莉娜**：从敌对到生死之交。莉娜本是来刺杀艾琳的，却被她的真诚打动。
- **凯恩 ↔ 莉娜**：互不信任→并肩作战。两人的价值观冲突与磨合是本作重要副线。
- **赛德 ↔ 艾琳**：导师与学生。赛德引导艾琳掌握时空魔法。
- **赛德 ↔ 凯恩**：互相尊重但理念不同。精灵的"长远眼光"与人类的"当下行动"之争。
