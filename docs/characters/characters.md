# 角色列表

## 主角

### 艾琳·星语（Erin Starwhisper）
- **性别**：女
- **年龄**：17岁
- **魔法派系**：时空系（极为罕见）
- **背景**：出生于北境小村庄，自幼能看见"未来的碎片"。被魔法学院发现后带入东陆。
- **性格**：内向但坚韧，善良却有决断力。

## 主要配角

### 凯恩·铁壁（Kane Ironwall）
- **性别**：男
- **年龄**：22岁
- **魔法派系**：元素系（专精火）
- **背景**：帝国骑士团最年轻的队长。奉命保护艾琳。
- **性格**：正直、责任感极强，有时过于刻板。

### 莉娜·影步（Lena Shadowstep）
- **性别**：女
- **年龄**：19岁
- **魔法派系**：暗影系
- **背景**：西荒刺客公会出身，因任务接近主角团后叛出。
- **性格**：表面冷漠，实则重情义。

### 赛德·古树（Thaddeus Oldtree）
- **性别**：男
- **年龄**：300+岁（精灵寿命）
- **魔法派系**：自然系
- **背景**：精灵族长老，知识渊博的学者。
- **性格**：睿智温和，偶尔犯迷糊。

## 反派

### 暗影之主（The Shadowlord）
- **身份**：不详
- **目标**：夺取源核力量，重塑世界
- **能力**：暗影系 + 炼金系的禁忌融合
